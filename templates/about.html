Acerca de Humanizador IA

Una aplicación web local para explorar herramientas de procesamiento de texto.

Funcionalidades Implementadas

  • Humanizar Texto: Aplica reemplazos simples para variar el texto.
  • Analizar Sentimiento: Clasifica el texto como Positivo/Negativo/Neutral usando VADER (si está disponible).
  • Resumir PDF: Extrae texto de PDF y genera resúmenes con Sumy/LSA.
  • Resumir Texto Pegado: Genera resúmenes de texto introducido directamente.
  • Buscar Término: Encuentra y resalta ocurrencias de palabras/frases.
  • Capturar Pantalla: Toma una captura de pantalla completa (si PyAutoGUI está disponible).
  • Limpiar Todo: Restablece la interfaz.

Tecnologías Clave

Backend: Python con Flask.

Procesamiento de Texto: NLTK (VADER, tokenizers, stopwords), Sumy (Resúmenes LSA), PyMuPDF (fitz) (Extracción PDF).

Captura de Pantalla: PyAutoGUI.

Formularios: Flask-WTF.

Frontend: HTML, Bootstrap 5, JavaScript (para AJAX).
